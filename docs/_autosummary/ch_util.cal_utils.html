Parameters:

  • arr (np.ndarray) ‚Äì 1D array containing the data. Arrays with more than one dimension are flattened.

  • bins (int or sequence of scalars or str) ‚Äì

    • If bins is an int, it defines the number of equal-width bins in rng.

    • If bins is a sequence, it defines a monotonically increasing

      array of bin edges, including the rightmost edge, allowing for non-uniform bin widths.

    • If bins is a string, it defines a method for computing the bins.

  • rng ((float, float)) ‚Äì The lower and upper range of the bins. If not provided, then the range spans the minimum to maximum value of arr.

  • no_weight (bool) ‚Äì Give equal weighting to each histogram bin. Otherwise use proper weights based on number of counts observed in each bin.

  • test_normal (bool) ‚Äì Apply the Shapiro-Wilk and Anderson-Darling tests for normality to the data.

  • return_histogram (bool) ‚Äì Return the histogram. Otherwise return only the best fit parameters and test statistics.

Returns:

  • results (dict) ‚Äì Dictionary containing the following fields:

  • indmin (int) ‚Äì Only bins whose index is greater than indmin were included in the fit.

  • indmax (int) ‚Äì Only bins whose index is less than indmax were included in the fit.

  • xmax (float) ‚Äì The data value corresponding to the centre of the indmax bin.

  • par ([float, float, float]) ‚Äì The parameters of the fit, ordered as [peak, mu, sigma].

  • chisq (float) ‚Äì The chi-squared of the fit.

  • ndof (int) ‚Äì The number of degrees of freedom of the fit.

  • pte (float) ‚Äì The probability to observe the chi-squared of the fit.

  • If return_histogram is True, then results will also contain the

  • following fields ‚Äì

    bin_centrenp.ndarray

    The bin centre of the histogram.

    bin_countnp.ndarray

    The bin counts of the histogram.

  • If test_normal is True, then results will also contain the following fields ‚Äì

    shapirodict
    statfloat

    The Shapiro-Wilk test statistic.

    ptefloat

    The probability to observe stat if the data were drawn from a gaussian

    andersondict

    The critical values of the test statistic.
